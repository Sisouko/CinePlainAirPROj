A modern, cinematic streaming platform landing page inspired by services like Netflix and Prime Video.
Built using HTML, CSS, and JavaScript, this project focuses on delivering a premium UI/UX experience with rich visuals, smooth layout, and engaging sections.

🚀 Live Concept

CinemaFes is a fictional streaming platform that allows users to explore:

Movies 🎥
TV Shows 📺
Trending content 🔥
Characters & actors 👤
Filming locations 🗺️
✨ Features
🎯 Core UI Sections
🔹 Navbar with logo, navigation links, search icon, and user avatar
🔹 Hero Section with cinematic background, tags, rating, and CTA buttons
🔹 Trending TV Shows horizontal scroll with ranking system
🔹 Golden Globe Banner premium highlight section
🔹 Series Section with ratings and progress bars
🔹 Interactive Map for filming locations (SVG-based)
🔹 Characters Section with circular profile cards
🔹 Family Section for kids content
🔹 FAQ Section with toggle functionality
🔹 Footer with links and social icons
🛠️ Technologies Used
HTML5 – Structure
CSS3 – Styling & animations
JavaScript – Interactivity (FAQ toggle, UI behavior)
Google Fonts – Montserrat & Poppins
